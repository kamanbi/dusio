# 조선 SD 장기 프로젝트 기획서
_Unreal Engine + Flutter 기반 모바일 전략 보드게임_

---

# 1. 프로젝트 개요

## 프로젝트명
```text
가칭: 조선장기 / Joseon Janggi
```

## 장르
```text
3D SD 전략 보드게임
턴제 장기 게임
```

## 핵심 컨셉

조선시대 분위기의 SD 캐릭터 기반 장기 게임.

기존 2D 장기와 달리:
- 모든 기물을 SD 3D 캐릭터화
- 공격/이동 애니메이션 존재
- 마지막 왕 제거 시 시네마틱 연출 제공
- 모바일 최적화 전략 게임

---

# 2. 개발 구조

## 전체 아키텍처

```text

↓

```
